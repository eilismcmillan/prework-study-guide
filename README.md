# Bootcamp Prework Study Guide


## Description

This web application provides a prework study guide for students to organise their bootcamp prework notes, and provide a central location to access information learned. The topics covered in this guide include HTML, CSS, Git, and Javascript.

## Installation

N/A

## Usage

This webpage is divided into sections, which are dedicated to the following topics; HTML, CSS, Git, and JavaScript. By consulting each section, you can review the specific notes made for this topic. For suggestions on what to study first, open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). A console panel should open either below or to the side of the webpage in the browser. There you will see a list of topics we learned from the prework along with a suggestion on which topic to study first.
